Introdução

JavaScript é uma plataforma, de linguagem de script. É uma linguagem pequena e leve, dentro de um ambiente host ( por exemplo, um navegador web ), JavaScript pode se conectar a objetos deste ambiente para permitir o controle programático sobre ele.

JavaScript contém uma biblioteca padrão de objetos, com as Arrays, Data, e Math, e um conjunto básico de elementos de linguagem como operadores, estruturas de controle, e afirmações. O JavaScript básico pode ser extendido para uma variedade de propósitos, o suplementando com ojetos adicionais, por exemplo:

  • A versão cliente do JavaScript extende a linguagem básica por suplementação de objetos de controle ao navegador e o seu Document Object Model (DOM). Por exemplo, as extensões da versão cliente permitem que uma aplicação colocar elementos em forma HTML e responder a eventos do usuario como clicks do mouse, formularios input, e navegação da pagina.
  • A versão server do JavaScript extende a linguagem basica para suplementar objetos relevantes para rodar JavaScript em um servidor. Por exemplo, as extensões da versão server permitem que as aplicações se comuniquem com a database, fornecendo informação continua de um lugar para outro da aplicação ou executando manipulações no servidor.

JavaScript e Java

JavaScript e Java são similares de algumas maneiras, mas fundalmentalmente diferente em outras. O JavaScript se assemelha ao Java mas o Java não tem um tipo estatico e verificação forte. JavaScript segue a maioria das expressões de sintax do Java, nomeadas convenções e construções básicas de controle de fluxo e por essa razão foi renomeado de LiveScript para JavaScript.

Em contraste com o Java um sistema de compilado de tempo construido por declarações, o JavaScript suporta a execução de um sistema baseado em um pequeno numero de tipos de data representados como numerico, boolean, e string. JavaScript tem um modelo baseado em prototipo em vez de um comum como modelo de objeto baseado em classe. o modelo baseado em prototipo fornece herança dinamica, isso é, herança pode mudar nos objetos individuais. JavaScript também suporta funções sem qualquer requisição de declarativas especial. Functions podem ser propriedades de objetos, executando metodos digitados.

JavaScript é uma linguagem muito livre, se comparada ao Java. Você tem que declarar todas as variaveis, classes, e metodos. Você nao tem que se preocupar se os metodos são publicos, privados, ou protegidos, e você não tera de implementar interfaces, variaveis, parametros, e funções não são explicitamente digitadas.

Variáveis

Você usa as variáveis como nomes simbolicos para os valores en sua aplicação. Os nomes das variáveis, são chamados de indentificadores, conforme certas regras.

Um identificador JavaScript deve começar com uma letra, underline (_), ou sifrão de dolar ($), os caracteres subsequentes podem ser digitos (0-9). Pois JavaScript é bem sensitivo, letras incluindo os caracteres "A" entre "Z"(maiusculas) e os caracteres "a" entre "z"(minusculas.)

Você pode usar ISO 8859-1 ou Unicode de letras como ü em identificadores. Você pode também usar o Unicode como espaço em caracteres identificadores. Alguns exemplos de nomes são Number_hits, temp99, and_name.

Declarando Variáveis

Você pode declarar variáveis de três maneiras:

Com a palavra chave var por exemplo;

var x = 42.

Essa sintaxe pode ser usada para declarar ambas variações locais e globais.

Ou simplesmente atribuindo ao valor;

x = 42.

Esse é sempre usado para uma variavel global.

Usando a palavra chave let, por exemplo;

let y = 13.

Essa variavel pode ser usada, para declarar um variavel local.
